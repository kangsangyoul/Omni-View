# Omni-View

## 실행 방법

### 백엔드
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### 프론트엔드
cd frontend
npm install
npm start

## 접속
- http://localhost:3000 (프론트엔드)
- http://localhost:8000 (FastAPI)

## 기능
- 실시간 이상행위 감지 및 자동 알림 (SQLite DB 영구 저장)
- 실시간 대시보드/그래프/카드는 SSE로 실시간 갱신
- 알림은 SSE(Server-Sent Events)로 실시간 스트리밍
- 섹시한 UI, 반응형 디자인, 고급 네비/구조
- 알림을 CSV로 다운로드 가능
