# Omni-View

## 실행 방법

### 백엔드
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### 프론트엔드
cd frontend
npm install
npm start

## 접속
- http://localhost:3000 (프론트엔드)
- http://localhost:8000 (FastAPI)

## 기능
- 실시간 이상행위 감지, 자동 알림, 실시간 대시보드/그래프/카드
- 섹시한 UI, 반응형 디자인, 고급 네비/구조
